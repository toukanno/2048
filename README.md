# 🔢 2048

4×4のグリッドで同じ数字のタイルを合体させて「2048」を目指すパズルゲーム。

## 遊び方
- **↑↓←→ / WASD** — タイルをスライド
- スマホはスワイプ操作に対応
- **「戻す」ボタン** — 1手だけ元に戻せる

## ルール
1. 全タイルが同じ方向にスライドする
2. 同じ数字のタイルがぶつかると合体して2倍になる
3. 2048のタイルを作ったらクリア（その後も続けられる）
4. 動けなくなったらゲームオーバー

## 特徴
- アンドゥ機能（1手戻す）
- ベストスコア記録
- 滑らかなアニメーション
- 2048達成後も続けてより高いスコアを狙える

## 技術スタック
HTML / CSS / Vanilla JavaScript
